# Stock Trading Platform

A Java-based console application that simulates a basic stock trading environment. This project allows users to view market data, buy and sell stocks, and track portfolio performance using Object-Oriented Programming concepts.

---

## Features

- View live-style market stock data
- Buy stocks
- Sell stocks
- Track portfolio balance
- Store transaction history
- Console-based interactive menu
- Object-Oriented Programming implementation

---

## Technologies Used

- Java
- ArrayList
- Scanner Class
- OOP Concepts

---

## OOP Concepts Used

- Classes and Objects
- Encapsulation
- Constructors
- Methods
- Collections using ArrayList
